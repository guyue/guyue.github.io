阶乘函数

阶乘函数，在数学中表示为n!，表示为1到n之间的所有整数的连乘积。

  • 递归实现
  • 迭代实现
{% highlight c linenos %} int Fact(int n){ if(n==0){ return 1; } else{ return n * Fact(n-1); } } {% endhighlight %}
{% highlight c linenos %} int Fact(int n){ int i, product; if(n==0){ return 1; } for(i=1, product=1; i<=n; i++){ product *= i; } return product; } {% endhighlight %}
